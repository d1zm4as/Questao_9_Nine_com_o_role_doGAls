# Questao_9

Java project using GALS artifacts for language analysis.

## Main components

- `QUESTAO_A3(ATT).gals`: grammar/specification input.
- `src/GALS`: generated or supporting lexical/syntactic/semantic classes.
- `src/Questao_9/Questao_nine.java`: main project entry point.
- `build.xml`: Ant build file.

## Build

```bash
ant
```
